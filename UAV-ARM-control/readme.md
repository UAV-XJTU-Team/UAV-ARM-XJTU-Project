## How to start:

- without visual control module:

    catkin_make
    
    source devel/setup.bash
    
    rosrun UAV-ARM-control my_arm_control

 ###    then open a new terminal:

    source devel/setup.bash
    
    rosrun UAV-ARM-control space_to_move
    
   
- with visual control module:

    catkin_make
    
    source devel/setup.bash
    
    rosrun UAV-ARM-control my_arm_control
